# kokoc_test
test assignment for the python developer vacancy

### Склонировать проект
```
git clone git@github.com:bdwayne11/kokoc_test.git
```

### Установить зависимости
```
poetry install
```

### Запустить проект

```
cd testing
python manage.py migrate
python manage.py runserver
```
